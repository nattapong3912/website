MWCH Server

Open-source Survival Server for Mini World Block Art fully customizable. Currency, Pets, Weapons and many more things are waiting for you.

Getting Started

What is MWCH?

MWCH is a simple, but powerful plugin pack. It's the first Public Survival Server Project that adds new functionalities to the game, like the Currency System, Inventory Shop and the coming Player Level System. It's powered and maintained by the Mini World-Hispanos Community. Open-source and simple to edit and use in other Map Projects.

Easy to install and Edit

Get started by cloning source into GitHub account of your project. Then start editing it and make it yours! Remember that the project is Licensed by the MIT License, give it a read.

The Community is Important

All the changes we made are suggested by and for the Community in our Discord Server.

Too many Features!

We have a hundred of features, the most important ones are the Currency System, Pets, Inventory Shop, and Pacific Lobby System, help us to make more!
